Most Common Terminal Commands

File Operations

ls -la List all files with details
cd ~ Go to home directory
mkdir -p folder/subfolder Create nested directories
cp -r source dest Copy directory recursively
rm -rf folder Remove directory and contents
find . -name "*.js" Find files by name pattern

Git Commands

git status Check repository status
git add . Stage all changes
git commit -m "message" Commit with message
git push origin main Push to remote main branch
git pull origin main Pull latest changes
git log --oneline Show commit history

Process Management

ps aux | grep process Find running processes
kill -9 PID Force kill process
top Show running processes

Most Common SQL Queries

Basic Queries

SELECT * FROM users;
Select all records from users table 
SELECT id, name, email FROM users WHERE active = 1;
Select specific columns with condition 
SELECT COUNT(*) FROM users;
Count total records

Joins

SELECT u.name, o.total 
FROM users u 
INNER JOIN orders o ON u.id = o.user_id;
Inner join users and orders
